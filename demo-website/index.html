Style Methods

            
    
              
  • .foreground(color) - Set text color

  • .background(color) - Set background color

  • .bold() - Make text bold
    • 
              
  • .italic() - Make text italic
    • 
              
  • .underline() - Add underline
    • 
              
  • .padding(t, r, b, l) - Add padding
    • 
              
  • .margin(t, r, b, l) - Add margin

Border Methods

            
    
              
  • .borderStyle(name) - Set border style
    • 
              
  • .borderAll() - Enable all borders
    • 
              
  • .borderTop() - Enable top border
    • 
              
  • .borderBottom() - Enable bottom border
    • 
              
  • .borderLeft() - Enable left border
    • 
              
  • .borderRight() - Enable right border

Layout Methods

            
    
              
  • .width(n) - Set fixed width
    • 
              
  • .height(n) - Set fixed height
    • 
              
  • .alignLeft() - Align content left
    • 
              
  • .alignCenter() - Align content center
    • 
              
  • .alignRight() - Align content right
